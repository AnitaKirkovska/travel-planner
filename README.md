# Travel Planner ✈️

Turn your travel emails into trip checklists. Scans Gmail for flight confirmations, hotel bookings, and activity reservations — then structures everything into one view with pre-trip, mid-trip, and post-trip checklists.

## Install

```
assistant plugins install travel-planner
```

## What it does

- **Scan Gmail** — finds flight bookings, hotel confirmations, event tickets, restaurant reservations
- **Build checklists** — pre-trip (packing, travel docs), mid-trip (daily agenda), post-trip (receipts, expenses)
- **Reminders** — weekly Sunday preview, 24h pre-trip reminder
- **Trip dashboard** — "what trips do i have coming up?" shows everything with checklist progress

## Built-in templates

Comes with preset templates for concert/festival trips, standard vacations, business trips, and weekend getaways. The assistant adapts to the trip type — a concert trip gets earplug reminders and venue bag-policy checks, a business trip gets meeting syncing and expense tracking.
